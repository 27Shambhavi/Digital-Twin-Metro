# Repository Profile

## One-Sentence Description

A SUMO and AI-powered digital twin for simulating metro construction traffic disruption, predicting congestion, and evaluating mitigation strategies.

## One-Paragraph Description

AI Metro Digital Twin integrates OpenStreetMap, Eclipse SUMO, machine learning, deep learning, and Streamlit to analyze how metro construction affects urban traffic flow. It supports scenario simulation, dataset generation, feature engineering, congestion prediction, model comparison, and mitigation strategy evaluation for smart-city mobility research.
